The `version` command is used to get client build information.

## Usage
```
accio version
```

This command requires no argument.
It prints the current version of the Accio client binary.

## Exit codes
* `0`: Success.

## Examples
Get client build information:

```
$ accio version
Build version: 2.0.0-dev
```
