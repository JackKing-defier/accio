Accio is a scientific workflow management tool, with a strong emphasis on location privacy.

Learn more » Get started »

Simple yet powerful DSL

Experiments are defined with a user-friendly JSON syntax, still supporting more complex use cases such as parameter sweeps.

Batteries included for location privacy research

Accio is bundled with many pre-defined operators, allowing any privacy researcher to quickly begin experimenting with state-of-the-art algorithms.

Multi-tenant and collaborative

Many users can use the same Accio cluster. Experiments are easily accessible, allowing users to monitor progress and collaborate on results.

Extensible

Accio is built around a plugin system, making it to adapt to your very own architecture and requirements.
