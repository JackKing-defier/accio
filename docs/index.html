Why using Accio?

Simple yet powerful DSL

Experiments are defined with a user-friendly JSON syntax, while still supporting more complex use cases such as parameter sweeps.

Batteries included for location privacy

Accio allows to quickly get started by providing many built-in state-of-the-art location privacy algorithms.

Easy results analysis

Accio provides tools for quick insights into results, as well as sophisticated ways to export your data for further analysis.

Multi-tenant and collaborative

Several users can share the same Accio cluster and collaborate on their results.
